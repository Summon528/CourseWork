  7. Comparison & Conclusion

    • K-fold vs Holdout
      While both methods provide similiar result, I feel like K-fold is more consistent and more reliable because it uses more data. For example, while using holdout validation in the Iris dataset, we did not even test any Iris-Sentosa data because it tests less data than k-fold.

    • Smoothing vs No smoothing
      Interesting enough, the result in my experiment shows that it is more accurate if smoothing is not applied. I believe this is due to the fact that I assume log(0) = -1e9 and it seems to be more appropriate for this specific set.
